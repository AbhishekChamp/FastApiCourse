8. Working with files
9. Tasks
10. Async await, templates, middleware
11. Dependencies
12. OCR Application
13. Blog Site - ReactJS, FastAPI
14. Instagram - React JS, FastAPI, Android
15. Warehouse app - Microservices and Redis
